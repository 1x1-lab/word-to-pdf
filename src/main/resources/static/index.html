Word 转 PDF

上传 .docx 模板，填充 ${变量}，导出 PDF（顶部自动插入 Logo）。
模板里用 ${title}、${user.name} 等占位符。
键名与 ${...} 内的文本一致。值可以是字符串、数字、布尔值。
宽 cm
高 cm
只填一个 = 另一个按比例自动算。都留空 = 用配置默认值。
不传则使用系统默认 Logo。支持 PNG / JPG / GIF / SVG。
